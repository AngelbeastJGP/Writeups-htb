# Hack The Box Writeups

Writeups técnicos de máquinas de Hack The Box realizados con fines formativos.

## Enfoque
- Enumeración metódica
- Explotación controlada
- Escalada de privilegios
- Mitigación y buenas prácticas (visión ASIR)

Las soluciones no están pensadas para copiar, sino para entender el proceso.
